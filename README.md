# 🌤️ Weather by Arif

A modern, responsive, and user-friendly **Weather Application** that provides real-time weather information for cities around the world. The app fetches live weather data through a weather API and presents it with a clean and intuitive interface.

🌐 **Live Demo:** https://wheatherbyarif.netlify.app/

## 🚀 Project Overview

**Weather by Arif** is a web application built to demonstrate API integration, asynchronous JavaScript, and responsive frontend development. Users can search for any city and instantly view current weather conditions, temperature, humidity, wind speed, and other key weather metrics.

The project was developed to strengthen practical skills in working with third-party APIs and building real-world web applications.

## ✨ Features

* 🌍 Search weather by city name
* 🌡️ Real-time temperature and weather conditions
* 💧 Humidity and wind speed information
* ☁️ Dynamic weather icons and descriptions
* 📱 Fully responsive design for all devices
* ⚡ Fast and lightweight user experience
* 🔄 Live data fetched through a weather API

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript
* **API Integration:** Weather API
* **Responsive Design:** CSS Flexbox & Grid
* **Deployment:** Netlify
